# Vision UI Copy Plan

## Status

Completed.

## Goal

Reduce explanatory text in `/vision` and standardize terminal lines, transcript labels, and helper copy so the interface feels concise and action-oriented.

## Scope

- Included: prompt copy, transcript copy, status labels, placeholder text, and small terminal helper messages.
- Excluded: layout changes and backend routing changes.

## Steps

1. Audit the current copy for redundant or overly explanatory text.
2. Rewrite the terminal and transcript copy to be shorter and more direct.
3. Keep the minimum helpful guidance needed for first-use clarity.

## Validation

- `npm run type-check`
- `npm run build`

## Completion

- Terminal copy is shorter and more action-oriented.
- Voice and transcript labels are compact and consistent.
