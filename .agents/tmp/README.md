# Temporary Agent Work Products

This directory is for short-lived machine-readable files that support one active plan.

Use `.agents/templates/temp-work-product.template.yaml` for new files.

Rules:

- Every file must name its owning plan.
- Every file must state when it should be deleted.
- When the owning plan closes, delete the file, promote useful facts into durable docs, or mark it `archived` with a reason.
- Do not treat files in this directory as durable product or workflow source of truth.
