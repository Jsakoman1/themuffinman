# SideQuest Frontend

Vue 3 + TypeScript frontend for the `SideQuest` module inside `TheMuffinMan`.

## Commands

Run from `apps/sidequest/frontend`:
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run type-check`

## Main Responsibilities

The frontend currently provides:
- authentication screens
- dashboard and quest browsing flows
- quest detail and application detail views
- profile pages
- circles management
- admin management pages
- review submission and rating display

## Configuration

Set `VITE_API_BASE_URL` to point the frontend at a non-default backend URL.
